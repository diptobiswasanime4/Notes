Database System Utilities help [[Database Administrator]] manage the Database System.

Common utilities have the following functions:

1. Loading
	1. Used to load existing data files into the database
	2. Source file format and target data file structure are mentioned to the utility. Based on that the utility automatically reformats data, and stores it in the database
2. Backup
	1. Creates a backup copy of the database
	2. Incremental backups are often used. Complex but saves space
3. File reorganization
4. Performance monitoring
