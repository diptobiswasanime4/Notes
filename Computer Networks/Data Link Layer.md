Data Link layer is the 2nd layer of the [[OSI Model Layers]]

Data Link Layer is responsible for moving data (frames) from one node to another.
## Services

1. [[Framing]]
2. Physical addressing
3. Flow Control
4. Error Control
5. Access Control

Error Control is divided into:

1. Error Detection
2. Error Correction
